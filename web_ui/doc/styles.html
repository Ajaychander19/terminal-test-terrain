(static) generateGradient(stepNum)

Produces a gradient made of a given number of color for HexBin layers. The gradiant is created using RGB-hex from HSL (Hue-Saturation-Luminosity) conversion, by iterating over HSL values with H in [240, 0], over the given number of steps, and with S: 1.0, L: 0.5.
